# 💰 MyBudget – Personal Finance App UI/UX Design

## 📌 Overview

MyBudget is a personal finance mobile application designed to help users manage their daily expenses, track their budgets, and understand their spending habits through simple and intuitive interfaces.

The goal of this project is to create a clean, user-friendly, and visually appealing finance management experience.

---

## 🎯 Project Goal

The main goal of MyBudget is to make personal finance management simple and easy for users.

Users can:

- Create an account
- Login to their account
- Add and manage transactions
- Set and track budgets
- Monitor their expenses
- View spending insights
- Manage their profile

---

## ✨ Features

### 🔐 Authentication
- Sign Up
- Login
- Forgot Password

### 🏠 Dashboard
- View financial overview
- Track recent transactions
- Quick access to important features

### 💳 Transactions
- Add new transactions
- Manage income and expenses
- Categorize transactions

### 📊 Budget Management
- Set monthly budgets
- Track budget progress
- Monitor spending

### 📈 Insights
- Visualize spending patterns
- Understand financial habits
- Track expenses

### 👤 Profile
- Manage user profile
- Access account settings
